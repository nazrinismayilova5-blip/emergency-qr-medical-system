# Emergency QR Code Medical Information System

## Description
This project presents a secure web-based medical information system that uses QR code technology to provide fast access to critical patient emergency data for authorized medical staff.

## Problem
In emergency situations, patients may be unconscious or unable to communicate. Essential medical information such as allergies, blood type, chronic diseases, and emergency contacts is often unavailable, which can delay treatment and increase medical risks.

## Objective
To develop a secure, fast, and easy-to-use system that allows authorized medical personnel to access essential patient emergency data through QR codes.

## System Workflow
1. System verifies user login
2. Admin registers or edits patient data
3. System generates a unique token and QR code
4. Medical staff scans the QR code
5. Patient emergency data is displayed securely
